# TF-Gen
TF-Gen (Text File Generators) contains two python scripts. One of which generates username lists and the other generates password lists that can be paired with tools like Hydra for brute force attacks.

Prior to installation, ensure you have python installed in your terminal and it's up to date.

To install, simply type "git clone https://github.com/RavenTheBird789/TF-Gen" in your terminals command line

To run, simply type either "python3 UNGen.py" or "python3 PWGen.py" in your terminals command line
(Note: Prior to running, ensure that you use a text editor such as vim or nano to modify the python scripts so that there are values within the given lists that can be made into .txt files that aren't blank as both python scripts lists are blank by default.)
